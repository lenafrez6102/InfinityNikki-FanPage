# 🌸 Infinity Nikki Promo! 🌸

A fun and stylish promotional web page for the upcoming open-world game **Infinity Nikki**!

## Overview

This project is a whimsical HTML-based promo site themed around **Infinity Nikki**, a fashion-forward adventure game where players can explore a magical world, collect outfits, and participate in style battles.

The page is designed with playful colors, charming content, and embedded media to convey the spirit of the game.

## 🌟 Features

- 🩷 A pastel-themed background and colorful heading
- 📸 A centered image of Nikki, the main character
- 👗 Descriptions of magical outfit powers:
  - Floating like a bubble
  - Purifying enemies
  - Catching butterflies
- 🔗 External links to the official Infinity Nikki website
- 💌 Contact link via email for questions or tips on gameplay
- 🎬 Embedded official promo video via YouTube
- 🐱 Fun ASCII art footer and developer credits
